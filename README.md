# ClipboardHistory 剪贴板历史管理器

跨平台剪贴板历史管理工具，支持 Windows 和 macOS 系统。

## 功能特性

- 📋 **剪贴板监控** - 自动记录文字和图片
- 🔍 **搜索过滤** - 快速查找历史记录
- 📌 **置顶功能** - 重要内容置顶不会被清理
- 🗑️ **删除功能** - 手动删除不需要的记录
- ⚙️ **自动清理** - 支持 1/3/5/7 天自动清理
- 🚀 **全局快捷键** - 快速调出剪贴板历史

## 系统要求

### Windows
- Windows 10/11 (x64)
- 无需额外安装依赖

### macOS
- macOS 10.15 (Catalina) 或更高版本
- 支持 Apple Silicon (M1/M2/M3)

## 数据存储位置

| 系统 | 路径 |
|------|------|
| Windows | `D:\ClipboardHistory\` |
| macOS | `~/ClipboardHistory/` |

存储内容：
- `clipboard.db` - 数据库文件
- `images/` - 图片历史存储
- `settings.json` - 设置文件
- `error.log` - 错误日志

## Windows 使用说明

### 运行应用

1. 进入 `release\win-unpacked\` 文件夹
2. 双击 `ClipboardHistory.exe` 运行
3. 应用会在后台运行，系统托盘显示图标

### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + V` | 显示/隐藏主窗口 |

### 系统托盘

- **左键双击**: 显示主窗口
- **右键菜单**:
  - `Show` - 显示主窗口
  - `Exit` - 退出应用

### 主窗口操作

#### 查看历史记录
- 默认显示全部历史
- 点击 `文字` / `图片` 标签页切换类型
- 使用搜索框输入关键词搜索
- 选择日期范围过滤历史

#### 复制历史内容
- **双击**任意记录即可复制到剪贴板

#### 置顶记录
- 点击记录右侧的 `置顶` 按钮
- 置顶内容会显示在列表顶部
- 置顶内容不受自动清理影响
- 再次点击可 `取消置顶`

#### 删除记录
- 点击记录右侧的 `删除` 按钮
- 删除后无法恢复，请谨慎操作

#### 设置
- 点击右上角设置图标
- **自动清理天数**: 选择 1/3/5/7 天
- **开机自启动**: 勾选后开机自动运行
- 点击 `保存` 生效

## 构建说明

### 环境要求

- Node.js 18.x 或更高版本
- npm 8.x 或更高版本

### 安装依赖

```bash
npm install
```

### 构建应用

```bash
# 构建 Windows 版本
npm run build

# 构建 macOS 版本 (需要在 macOS 上运行)
npm run build:mac

# 同时构建两个平台
npm run build:all
```

### 输出目录

构建完成后，可执行文件位于:
- Windows: `release\win-unpacked\ClipboardHistory.exe`
- macOS: `release\mac-unpacked\ClipboardHistory.app`

## 常见问题

### Q: 为什么快捷键不起作用？
A: 请确保应用正在运行（系统托盘有图标），且没有其他程序占用该快捷键。

### Q: 历史记录丢失了？
A: 请检查自动清理设置，可能记录已超过保存天数被自动删除。置顶的记录不会被清理。

### Q: 如何完全卸载？
A:
1. 退出应用
2. 删除应用文件夹
3. (可选) 删除数据存储文件夹:
   - Windows: `D:\ClipboardHistory\`
   - macOS: `~/ClipboardHistory/`
