# 剪贴板历史管理器 - ClipboardHistory

## 1. 项目概述

- **项目名称**: ClipboardHistory
- **类型**: Windows 桌面应用程序
- **核心功能**: 记录用户所有复制内容（文字/图片），支持搜索、置顶、删除，按时间降序展示
- **目标用户**: Windows 用户，需要管理复制历史记录

## 2. 技术栈

- **框架**: Electron 32.x
- **前端**: HTML + CSS + Vanilla JS（简单直观，避免复杂框架）
- **数据库**: SQLite（better-sqlite3）
- **构建工具**: electron-builder
- **数据存储**: D盘固定路径 `D:\ClipboardHistory\`

## 3. UI/UX 规范

### 窗口结构
- **主窗口**: 单窗口应用，固定大小 800x600，可拖拽调整
- **系统托盘**: 关闭时最小化到托盘，右键菜单包含"显示"和"退出"
- **全局快捷键**: `Ctrl+Shift+V` 呼出主窗口

### 视觉规范
- **主色调**: 淡蓝色 `#E8F4FC`，强调色 `#4A90D9`
- **文字颜色**: 主文字 `#333333`，次要文字 `#999999`
- **卡片**: 白色背景 `#FFFFFF`，圆角 8px，轻微阴影
- **字体**: 系统默认字体，大小 14px
- **间距**: 基础间距 12px，卡片间距 8px

## 4. 功能规范

### 4.1 复制监控
- 监听系统剪贴板变化
- 区分文字和图片类型
- 文字直接存储
- 图片保存到 `D:\ClipboardHistory\images\`，数据库存储路径引用
- 去重：新复制内容与最近一条相同则不记录

### 4.2 数据展示
- Tab 切换：全部 / 文字 / 图片
- 搜索：支持按内容关键词搜索
- 日期过滤：显示最近 N 天的记录（N = 1/3/5/7）
- 按时间降序排列，置顶项始终在最前面

### 4.3 置顶功能
- 点击置顶按钮将条目移到最前
- 置顶内容不受自动清理影响
- 再次点击取消置顶

### 4.4 删除功能
- 点击删除按钮移除条目
- 图片同步删除文件系统中的文件

### 4.5 自动清理
- 根据设置的天数（1/3/5/7），自动清理非置顶的过期内容
- 启动时和定时检查（每小时）执行

### 4.6 设置功能
- 存储天数设置：1天 / 3天 / 5天 / 7天
- 开机自启动开关

### 4.7 双击复制
- 双击卡片内容，将内容重新复制到系统剪贴板

## 5. 数据结构

### 数据库表: clipboard_items
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 自增主键 |
| type | TEXT | 'text' 或 'image' |
| content | TEXT | 文字内容或图片路径 |
| created_at | INTEGER | 时间戳（毫秒） |
| is_pinned | INTEGER | 0=普通 1=置顶 |

## 6. 存储路径

```
D:\ClipboardHistory\
├── images\
│   └── {uuid}.png
└── db\
    └── clipboard.db
```
